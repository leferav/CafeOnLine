# Projeto Café On-line (Front-end)

## Rodar o projeto
1. Instale o Node.js (LTS)
2. No terminal, dentro da pasta do projeto:
   - `npm install`
   - `npm run dev`

## Rotas
- `/` Home
- `/lotes` Listagem
- `/lotes/novo` Cadastro

## Observação importante
O `BrowserRouter` já está configurado em `src/main.jsx` para evitar tela branca ao usar `NavLink`.
